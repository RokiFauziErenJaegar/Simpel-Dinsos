# Acceptable Use Policy (AUP) — SIMPEL DINSOS

> Dokumen kebijakan resmi untuk seluruh pegawai Dinas Sosial Kabupaten Pringsewu yang menggunakan SIMPEL DINSOS. Wajib dibaca, disetujui, dan ditandatangani sebelum onboarding.
>
> **Kontrol ISO 27001:** A.5.10 (Acceptable Use), A.6.1–A.6.6 (People Controls)

## 1. Tujuan

AUP ini menetapkan aturan, hak, dan tanggung jawab penggunaan Sistem Informasi Manajemen Pelayanan (SIMPEL DINSOS) yang dikelola Dinas Sosial Kabupaten Pringsewu.

## 2. Cakupan

Berlaku untuk:
- Seluruh pegawai Dinas Sosial Kabupaten Pringsewu (PNS, PPPK, honorer)
- Operator Pekon/Kelurahan yang diberi akses
- Vendor pihak ketiga yang diberi akses (terikat juga pada DPA)
- Kepala Dinas dan jajaran struktural

## 3. Akun & Autentikasi

### 3.1 Penggunaan Akun
- Akun bersifat personal — **TIDAK BOLEH** dibagikan ke siapa pun, termasuk rekan kerja atau atasan.
- Pelanggaran berbagi akun dapat dikenakan sanksi disipliner sesuai PP 94/2021.
- Setiap login tercatat di audit log; pegawai bertanggung jawab penuh atas seluruh aktivitas dari akunnya.

### 3.2 Password
- Minimal 12 karakter, gabungan huruf besar/kecil/angka/simbol.
- Tidak boleh sama dengan password 5 generasi terakhir.
- Wajib diganti tiap 90 hari (akan ada notifikasi dari sistem).
- **DILARANG** menulis password di kertas, sticky note, atau Notes HP.
- Pakai password manager (Bitwarden / 1Password / KeePass) untuk menyimpan.

### 3.3 2FA Wajib
- Role **Admin, Kadis, Sekretaris** WAJIB mengaktifkan 2FA TOTP.
- Recovery codes wajib disimpan di tempat aman (NOT screenshot di HP).
- Kehilangan akses 2FA: hubungi DPO untuk reset, dengan verifikasi tatap muka.

### 3.4 Logout
- Logout setelah selesai bekerja.
- Layar dikunci (`Win+L` / `Cmd+Ctrl+Q`) saat meninggalkan meja walau sebentar.
- Session timeout sistem 120 menit.

## 4. Penggunaan Data Pribadi PPKS

### 4.1 Prinsip Minimisasi (UU PDP Pasal 16)
- **HANYA** akses data PPKS yang relevan dengan tugas Anda.
- Setiap akses tercatat di `data_access_logs` dan dapat di-audit Inspektorat.
- "Iseng" melihat data tetangga, teman, atau public figure = pelanggaran berat.

### 4.2 Larangan Penyebaran
- **DILARANG** screenshot NIK, KK, atau dokumen PPKS untuk diposting di WA, FB, IG, atau media lain.
- Sanksi: pelaporan ke Polri sesuai UU ITE Pasal 32 (penyalahgunaan data elektronik).
- Lihat juga UU PDP Pasal 65 (penyebaran tanpa izin = penjara 4 tahun + denda 4 miliar).

### 4.3 Komunikasi dengan Warga
- Untuk verifikasi: pakai call center resmi 0822-6986-7911 (terekam) — bukan WA pribadi pegawai.
- Hindari minta foto KTP/KK lewat WA pribadi.
- Untuk konfirmasi pengajuan, gunakan fitur "Chat dengan Pemohon" di Filament admin (tercatat).

## 5. Penggunaan Berkas & Perangkat

### 5.1 Berkas Sensitif
- Berkas KTP/KK/foto PPKS hanya boleh diakses via fitur "Download Berkas" di sistem.
- **DILARANG** memindahkan ke USB, email pribadi, Google Drive personal, atau cloud lain.
- Untuk kebutuhan rapat/cetak: gunakan fitur ekspor sistem (yang tercatat) atau cetak dari komputer dinas.

### 5.2 Perangkat Dinas
- Hanya install software dari daftar yang disetujui Diskominfo.
- **DILARANG**: software bajakan, P2P (uTorrent), VPN gratis tidak resmi, browser ekstensi tidak dikenal.
- Antivirus wajib aktif dan ter-update.

### 5.3 BYOD (Bring Your Own Device)
- Akses dari HP pribadi: pakai PWA via browser, jangan login Filament admin dari HP pribadi.
- Akses dari laptop pribadi: hanya jika sudah terdaftar di MDM Diskominfo.
- WFH: wajib pakai VPN dinas + 2FA wajib aktif.

## 6. Etika Online & Sosial Media

### 6.1 Citra Lembaga
- Saat aktif di sosmed (FB, IG, TikTok), pegawai yang mengaitkan diri dengan Dinsos wajib menjaga citra lembaga.
- Hindari komentar politis, SARA, atau yang merendahkan kelompok rentan (PPKS, disabilitas, lansia).

### 6.2 Pengaduan
- Jika menemukan masalah sistem: laporkan ke DPO via call center internal.
- **DILARANG** publish bug, exploit, atau celah keamanan ke media publik tanpa izin tertulis Kadis.

## 7. Pelanggaran & Sanksi

| Pelanggaran | Sanksi |
|---|---|
| Berbagi password / akun | Teguran tertulis + reset akses |
| Akses data PPKS tanpa kebutuhan tugas | Peringatan keras + audit menyeluruh |
| Penyebaran data PPKS keluar sistem | Pelaporan ke Polri + sanksi disipliner PP 94/2021 |
| Bug exploit untuk keuntungan pribadi | Pemberhentian tidak hormat + pidana |
| Pemalsuan disposisi/surat | Pemberhentian + pidana pemalsuan dokumen |
